# ProjetoEngenhariaSoftware
istema de gerenciamento de chamadas de serviço para um provedor de internet. O sistema vai ter um perfil de cada funcionário, seu respectivo carro, e suas chamadas abertas criadas pelo supervisor. Além disso, haverá o historico das chamadas realizadas por cada funcionário, que podem ter diferentes tipos (instalação, manutenção, corte), com as datas, horas e endereços.
